# Argument Generator

A web-based tool to generate structured arguments for group discussions and debates.

## Features
- Generates arguments for given topics
- Useful for GD, debates, and interview prep

## Tech Stack
- Python
- Flask
- HTML, CSS

## How to Run
1. Install requirements
2. Run `python app.py`
3. Open browser at `localhost:5000`
